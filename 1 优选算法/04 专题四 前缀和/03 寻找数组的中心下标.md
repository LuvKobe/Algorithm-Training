### 2. 算法思路

从中心下标的定义可知，除中心下标的元素外，该元素左边的「前缀和」等于该元素右边的「后缀和」。

- 因此，我们可以先预处理出来两个数组，一个表示前缀和，另一个表示后缀和。
- 然后，我们可以用一个 `for` 循环枚举可能的中心下标，判断每一个位置的「前缀和」以及「后缀和」，如果二者相等，就返回当前下标。

### 3. 算法代码

**算法实现：**
```cpp
class Solution 
{
public:
    int pivotIndex(vector<int>& nums) 
    {
        int n = nums.size();
        vector<int> f(n), g(n); // 默认初始化为0

        // 1. 预处理【前缀和】数组以及【后缀和】数组
        for (int i = 1; i < n; i ++)
            f[i] = f[i - 1] + nums[i - 1];
        for (int i = n - 2; i >=0; i --)
            g[i] = g[i + 1] + nums[i + 1];
        
        // 2. 使用两个数组
        for (int i = 0; i < n; i ++)
            if (f[i] == g[i])
                return i;
        return -1;
    }
};
```
